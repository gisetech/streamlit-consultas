# Painel de Consultas Médicas — Streamlit

Este projeto é uma aplicação interativa desenvolvida em **Python + Streamlit + Plotly + Pandas**.  
O painel exibe **consultas médicas simuladas** a partir de um arquivo CSV (`consultas.csv`), com filtros por **data e unidade**, gráficos interativos e indicadores de desempenho (KPIs).


```
python3 -m venv .venv
source .venv/bin/activate

pip install --upgrade pip
pip install -r requirements.txt

streamlit run dashboard.py
```
